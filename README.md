# Histology ROI Quantification

Standardized contrast normalization and interactive ROI quantification for fluorescence microscopy images. Built for TRAP2-Ai14 (tdTomato/DAPI) `.nd2` files but generalizable to any multi-channel fluorescence data.

## Tools

### `process_all.py` — Batch Contrast Normalization
Two-pass processing of `.nd2` microscopy files across multiple animals:
1. **Pass 1**: Computes global percentile-based contrast limits (0.5th–99.5th) across all files
2. **Pass 2**: Applies consistent scaling to produce standardized output images

Outputs per-channel images (`red_color/`, `blue_color/`, `merged/`, grayscale) to `output2/` folders.

```bash
cd /path/to/experiment_folder
python process_all.py
```

### `process_nd2.py` — Single-File Processing
Process individual `.nd2` files with fixed scaling factors. Useful for quick previews.

```bash
python process_nd2.py Slide1-1_Region*.nd2
```

### `roi_quantify.py` — Interactive ROI Quantification (Streamlit)
Web app for comparing fluorescence expression across animals:
- **Tabs per animal** with multi-image selection
- **Multiple ROIs per image** (up to 6) with color-coded overlays
- **Percentage-based ROI positioning** — works with any image dimensions
- **Automatic statistics**: per-animal summary, one-way ANOVA, Kruskal-Wallis, pairwise t-tests
- **CSV export** of all measurements
- **Bar chart** comparison

```bash
cd /path/to/histology
streamlit run roi_quantify.py
```

## Setup

```bash
pip install -r requirements.txt
```

## Data Format
Expects Nikon `.nd2` files with two channels:
- Channel 0: 555nm (tdTomato)
- Channel 1: 395nm (DAPI)

Directory structure:
```
experiment_folder/
├── Animal-1/
│   ├── *.nd2
│   └── output2/        # generated by process_all.py
├── Animal-2/
│   └── ...
└── process_all.py
```

## Notes
- Handles extremely variable image sizes (tested with 3800×23000 strips to 28000×8600 panoramas)
- ROI quantification maps display coordinates back to full resolution
- `process_all.py` subsamples pixels (every 4th) during Pass 1 to manage memory on large datasets
